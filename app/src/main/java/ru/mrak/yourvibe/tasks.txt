    - create
    - editable                  - пока не реалзовано
    - vote
        - voted
            - agree
            - disagree
    - waiting                   - момент до которого можно голосовать. Пока не реализовано
    - result                    - результат может быть объявлен в период между окончанием голосования и датой окончания (В текущем варианте в любой момент)
        - true
        - false



Экраны:
        - Вайбы (
            Отображаются все вайбы за которые можно проголосовать, но за которые пользователь еще не проголосовал
            Отображаются в порядке создания
        )
        - Мои вайбы (
            Отображаются все вайбы созданные пользователем в следующих группах:
                Сначала вайбы срок которых закончился
                Затем ожидающие вайбы
                Затем вайбы с готовым результатом
            внутри групп вайбы сортируются в порядке создания
        )
        - Проголосованные вайбы (
            Отображаются вайбы за которые пользователь проголосовал, но которые на которые еще не было результата
            Вайбы срок которых вышел помечаются
        )
        - История голосования (
            Отображаются вайбы за которые пользователь проголосовал, на которых уже есть результат
            Должно отображаться:
                совпал ответ пользователя с результатом или нет
                в расширенном варианте детально, как проголосовал пользователь и кокой результат (?)
                статистика по проголосовавшим
        )
